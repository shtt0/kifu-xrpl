## ローカル実行

1. プロジェクトをクローンします:
   git clone https://github.com/shtt0/kifu-xrpl.git
   cd kifu-xrpl

2. 必要なパッケージをインストールします:
   npm install

3. ローカルサーバーを起動します:
   npm run dev

4. ブラウザで以下のリンクにアクセスします:
   http://localhost:3000/
